# SCPI Extra Credit Project for ECE 2k7 @ Purdue
- Note: LLMs were used to create plotting code, debugging, and for some code modifications/refactors.

## Project Details
- `uv` was used to create the virtual environment and to run the scripts.
- The `scripts` directory contains all the code for steps 3 and 4.
- The `plots` directory contains all the plots generated from steps 3 and 4 and an ideal XY mode plot for step 4.
- The `experiment_proof` directory contains images and screenshots, which all show that I actually did the experiments. I forgot to take a picture of the experiment setup for step 3.2, however.
- `sketch_apr22a/sketch_apr22a.ino` contains a mini SCPI parser; this script was deployed to an Arduino Mini.
- An Analog Discovery 2 from `ECE 270`'s lab kit was used to provide power to the Arduino and for generating the waveforms.
